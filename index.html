Notable Things

I'm really just adding this here for filler, because I feel like the front page should have more content, but I don't know what to put. I'm going to use this space to discuss DuckDuckGo and Firefox.

Duck Duck Go

Duck Duck Go is a search engine designed around privacy. There are no advertising trackers built in to the site. The ads that show up on Duck Duck Go are content-based ads and not behavior based ads. I've recently switched from using google search to DuckDuckGo, and have not experienced any substantial issues trying to find things. If you run into an issue, however, you can preface your search with "!g" and it will automatically bring up the Google results for your query.

Bang! Commands

This gets me to my next point about DuckDuckGo. Their innovative Bang! commands allow you to use the web in a whole different way. There are over 11,000 search shortcuts where you can interact with other sites directly from the DuckDuckGo search bar. For example, if you search "!a Dell XPS 15", it will automatically bring you to the search results on amazon for "Dell XPS 15".

The coolest usage of this feature, in my opinion is automatic translation. If there is some text that you want to translate to english (for example), you can preface your search with "!gten" and it will bring you to google translate for whatever succeeded "!gten" (to illustrate this, go try it yourself! Search on DuckDuckGo: "!gten Ciao Tutti!").

Instant Answers

The other cool feature of DuckDuckGo is their Instant Answers feature. Google has a similar feature, but in my opinion, DuckDuckGo does it better because you can do things like generate a random name, generate a random 12 character password, decode base64 encrypted text.

Firefox

Mozilla Firefox is a web browser, designed around data privacy and data protection. By default, Mozilla Firefox blocks different trackers that cause ads to show up for "new laptops" in your facebook feed, after you were just searching google hours ago for new laptops. To demonstrate this, look at the four trackers that get blocked from The New York Times just simply by using Firefox.

The other benefit of Firefox, over other browsers that have tracking-blocking-features (like brave) is that it runs it's own web engine. There are three major browser engines in 2020; Chromium (Google), Webkit (Apple), and Gecko (Mozilla). We used to have one from Microsoft, which used to run Microsoft Edge, but Microsoft recently redesigned Edge to run on the Chromium engine, so the Microsoft engine will essentially die off.

As annoying as it may be to have to develop some things specifically for different browser engines (-moz- or -webkit- prefixes for attributes in css), having a diverse set of browser engines helps contribute to the notion of a "free and open internet". When the power about how to view the web is distributed amongst different people, we can avoid a scenario where one company or organization gets to dictate how people experience the internet.
